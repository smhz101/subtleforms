## 3) API & response contracts рџ”—
- REST namespace: `subtleforms/v1`.
- Standard API shape: server responses use `{ data, meta }`; `resources/admin/data/apiClient.js` unwraps `.data` so components get the payload directly.
- Pagination is surfaced on arrays using `_pagination` / `_meta`.
- When changing request payloads or responses, update server `Schemas` and client expectations.

## 4) Focus area: Creating forms (common failure & fixes) рџ§Ғ
- Flow: UI calls `POST /forms` to create a form (title + optional metadata), then saves schema via `POST /forms/:id/schema.`
- Common failure: `Validation failed` for `config` or `schema` with message "Must be an object". This usually means:
  - The client sent an empty string for the field, or
  - The client sent a JSON string instead of an object.
- Mitigations added:
  - Server now attempts to decode stringified JSON for `config`/`schema` and treats empty strings as absent.
  - `Schemas::FORM_CREATE` now allows an optional `schema` (map:FormSchema).
  - CreateForm flow: the client creates the form first and then posts schema вЂ” schema failures are logged and non-blocking.

Triage checklist:
1. Inspect Network в†’ `POST /forms` payload and response; review `meta.fields` for validation error details.
2. If `config`/`schema` errors appear, check payload shape: ensure these fields are objects (not strings).
3. Check PHP debug logs for `SubtleForms:create_form - raw input:` and `- validated:` lines (we log input + validated values when enabled).
4. If schema save fails after creation, check the `/forms/{id}/schema` response and server logs.

## 5) Troubleshooting quick wins рџ› пёЏ
- 404 on `wp-json/subtleforms/v1/*`: check plugin activation, permalinks, and correct local URL (Valet host).
- 401 REST responses: verify admin session or that the `X-WP-Nonce` is present.
- Empty list with non-zero `total`: likely repository SQL is filtering columns or returning empty rows вЂ” inspect repository functions and SQL.
- Frontend type errors (e.g., `.filter is not a function`): inspect the API response shape (array vs object wrapper).
